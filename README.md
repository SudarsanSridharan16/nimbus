# nimbusとは

Nimbus は、SIerによるSIerのためのSI部品ライブラリです。
部品は、サービスと呼ばれるライフサイクルを持ったJavaアプリケーションで、JMXのMBeanとして実装する事で、JMXからも管理できます。
基盤としてDIコンテナを持ちます。 XMLの定義ファイルによって、ホスティングするサービスを定義し、サービス間の依存関係や、サービスへのインジェクションを行います。
